### Remove Duplicate Objects 

You are given an array of objects. Remove duplicate objects based on the id. 

```js
const users = [ 

{ id: 1, name: "John" }, 

{ id: 2, name: "Jane" }, 

{ id: 1, name: "John" }, 

{ id: 3, name: "Mike" } 

]; 
```

Expected Output 
```js
[ 

{ id: 1, name: "John" }, 

{ id: 2, name: "Jane" }, 

{ id: 3, name: "Mike" } 

] 
```
