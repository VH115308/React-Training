## Multiple-Choice Quiz Application 

#### Problem Statement Build a fun quiz with 15 questions (you can hardcode them). 

- Must-Have Features 
    1. Start screen with “Start Quiz” button 
    2. One question at a time with 4 answer options 
    3. Highlight correct/incorrect feedback after each answer. 
    4. Progress bar (Question 3 of 10) 
    5. Final score screen with percentage and “Restart” button 
    6. Timer per question (optional but cool) 


- Bonus Features 

    1. Confetti on score > 80% 
    2. Store high score in localStorage
